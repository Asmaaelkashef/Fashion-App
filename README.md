# Fashion Store 🛍️

تطبيق موبايل لبيع الملابس، الإكسسوارات، والأحذية.

## المميزات ✨

* عرض المنتجات (ملابس، شوزات، إكسسوارات).
* تفاصيل المنتج (صورة، وصف، سعر).
* إضافة المنتجات لعربة التسوق.

## التقنيات 🛠️

* Flutter
* Dart

## طريقة التشغيل ▶️

```bash
git clone <repo-url>
cd fashion-store
flutter pub get
flutter run
```
